# simulation – Test-Prototyp

Eigenständige Single-File-HTML-App zum Entwerfen und Austarieren des Bretts.

- Lädt eine **JSON-Datei von außen** (Brettaufbau, Felder pro Bereich, Kartenfelder,
  Würfelanzahl, Kartenstapel) und kann den Stand wieder als JSON exportieren.
- Spielt **Computer gegen Computer** (2–4 Spieler), automatischer Ablauf.
- Einzelnes Spiel sichtbar abspielen mit regelbarer Geschwindigkeit.
- **Turbo-Modus:** viele Partien ohne Animation durchrechnen.
- Auswertung: durchschnittliche Spieldauer, Siege je Startposition,
  kürzeste/längste Partie.

Noch nicht implementiert – wird als nächster Schritt gebaut.
